# Contributing

This is a knowledge-base repository — it contains technical documentation,
experimental data, reasoning traces, and memory snapshots. Contributions welcome:

- **Corrections to technical content**: open an issue or PR with the fix.
- **Additional experimental data**: if you reproduce a run and have additional
  findings, open an issue — we may link to your results or merge a summary.
- **Translations**: the report is in English; translations to other languages
  are welcome.
- **Reproducibility reports**: if you follow `docs/10_reproducibility.md` and
  hit issues, open an issue with system specs + error output.

This repo does NOT accept code contributions (it is documentation-only).
For software contributions, see [mdk-orchestra](https://github.com/nothingbugger/mdk-orchestra).
